# firestore - temples 
四国八十八か所のお寺の情報をfirestoreにインポートするコード
## Requirements
### HomeBrewでのインストール方法
```
$ brew install python3
$ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
$ python3 get-pip.py
$ pip install -r requirements.txt
```
